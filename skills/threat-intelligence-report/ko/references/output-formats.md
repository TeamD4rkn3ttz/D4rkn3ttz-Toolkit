# 결과물 형식과 문체

## 공통 문체

- 결론부터 쓰되 근거보다 강하게 단정하지 않는다.
- 짧고 직접적인 문장을 사용한다.
- `확인됐다`, `관찰됐다`, `주장했다`, `추정된다`, `확인되지 않았다`를 구분한다.
- 비판적인 내용도 조롱이나 인신공격 없이 검증 가능한 사실로 쓴다.
- 날짜는 절대 날짜와 타임존을 쓴다.
- 링크는 해당 주장을 뒷받침하는 문장 가까이에 둔다.

## 빠른 분석 요약

1. 한 줄 결론
2. 확인된 사실 3~5개
3. 미확인·주의점
4. 영향과 권고
5. 출처
